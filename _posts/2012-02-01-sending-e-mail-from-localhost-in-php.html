--- layout: post title: Sending e-mail from localhost in PHP categories: - PHP - Tips - Tools tags: [] status: publish type: post published: true meta: aktt_notify_twitter: 'no' _edit_last: '1' --- If you don't have access to a SMTP server this is what you need: http://smtp4dev.codeplex.com/ Windows 7/Vista/XP/2003/2010 compatible dummy SMTP server. Sits in the system tray and does not deliver the received messages. The received messages can be quickly viewed, saved and the source/structure inspected. Useful for testing/debugging software that generates email. If you do have access to a SMTP server all you need to do is update the servername  and smtp_port  in your php.ini file: 
SMTP = smtp.myserver.com
smtp_port = 25

For details on php mailing functions check out the manual.
